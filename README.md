# Traffic Accident Analysis & Prediction

## Overview
This project analyzes traffic accident data using machine learning techniques to identify patterns and predict accident severity.

## Techniques Used
- Data Cleaning & Preprocessing
- Feature Engineering
- K-Means Clustering (Descriptive Analysis)
- Decision Tree (Predictive Modeling)

## Key Insights
- Accident severity is influenced by weather, lighting, and number of casualties
- Clustering revealed different risk-level accident groups
- Model achieved ~78% accuracy in predicting accident severity

## Tools
- Python (Pandas, Scikit-learn)
- Jupyter Notebook
